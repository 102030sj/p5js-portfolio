과제 3 : Interactive Character

작품 설명

※ 키보드 조작 전 작품 화면을 한 번 클릭해주세요.

사용자의 입력에 따라 캐릭터의 상태가 변화하는 인터랙티브 작품이다. 키보드와 마우스를 활용하여 캐릭터의 외형을 변경하거나 이동시킬 수 있다.


조작 방법

🖱️ 마우스 왼쪽 클릭
→ 의상이 벗겨지며 상태 변화

⌨️ H 키
→ 머리 색상 변경

⌨️ C 키
→ 의상 색상 변경

⌨️ R 키
→ 기본 상태로 초기화

⬅️➡️⬆️⬇️ 방향키
→ 캐릭터 이동
